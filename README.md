# rpi2w-ups-portal

> 🚧 В стадии активной разработки

Веб-портал управления для устройства на базе Raspberry Pi 2W со встроенным ИБП (UPS).

Позволяет управлять устройством через браузер — даже когда оно не подключено к домашней сети, автоматически поднимая собственную точку доступа Wi-Fi.

## О проекте

Устройство на базе RPi 2W не всегда самостоятельно подключается к сети. Портал решает эту проблему: при отсутствии подключения RPi поднимает hotspot, пользователь подключается к нему с телефона или ноутбука и через веб-интерфейс настраивает Wi-Fi, следит за состоянием ИБП и управляет нагрузкой.

## Возможности (в разработке)

- Мониторинг ИБП — напряжение, ток, заряд батареи, статус сети 220 В
- Управление нагрузкой — временное отключение с настраиваемым таймером
- Настройка Wi-Fi подключения через браузер
- Автоматический fallback на hotspot при отсутствии сети
- Конфигурация через стандартный Linux-конфиг `/etc/rpi2w-portal/main.conf`
- GPIO пины настраиваются через конфиг без правки кода

## Стек

- **Python / Flask** — бэкенд портала
- **Gunicorn + Nginx** — продакшн-деплой
- **NetworkManager (nmcli)** — управление Wi-Fi
- **INA219** — мониторинг питания по I²C
- **GPIO** — управление реле и датчик AC
